What is the Internet?
A bunch of HTML documents connected via links being read by computer web browsers.

What are the major elements of the web?

What is HTML?
HyperTest Markup Language.
Text is what you see.
Markup: changes the look of what you see.

How does HTML Markup work?
It uses tags to give instructions to the computer.< >

What types of tags are there?
There are inline and block tags.
Examples of inline tags include: Anchor, EMphasis, BReak, SPAN, STRONG, IMaGe.
Examples of block tags include: Paragraph, Div and FORM.
Most tags have an opening and closing tag. But some tags don't have a closing tag. They are referred to as void tags.

What is an element?
Everything within a set of opening and closing tags.

What is an attribute?
It's a property of an HTML element.
